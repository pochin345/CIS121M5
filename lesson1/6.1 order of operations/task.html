Order of operations

In computing, as in math, multiplication and division
are performed before addition and subtraction.

So 3 + 5 * 0 is 3 not 0 as since 5 * 0 is performed first.
A lot of people don't understand this

But code within parentheses is performed first,
which a lot of people do understand

So when mixing multiplication and division with addition and subtraction,
it's good form to use parenthesis, and not to rely on anyone reading your code to understand order of operations

So code 3 + 5 * 0 as 3 + (5 * 0), if you want the normal order of operations. They are both the same, but the latter is more understandable
