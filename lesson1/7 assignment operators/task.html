Assignment Operators

So we already know the basic assignment operator =
This assigns the value on the right hand side to the variable on the left

But there are others.
We'll discuss two of these:

  • + = increases the variable on the left by the value on the right
  • - = decreases the variable on the left by the value on the right

Note There should be no space between
the + and the = or between the - and the =.
There is one in the definittion above just to make it more legible

So if cost = 10

  • cost += 2 makes cost 12
  • cost -= 4 makes cost 8
