Comparison Operators

Now we're going to talk about a new variable type: Boolean
Boolean variable can have only two values: True or False.

Boolean variables are the result of comparisons
Here are the comparison operators:

==the operands are equal
!=the operands are notequal
>=the left operand is
greater than or equal the right operand
<=the left operand is
less than or equal the right operand
>the left operand is
strictly greater than the right operand
>the left operand is
sytictly less than the right operand

Numeric comparisons

Numeric comparisons compare
the value of the two operands.

String Comparisons

While numeric comparisons are pretty straightforward,
string comparisons are a little tricky.

String comparisons take place character by character,
using ASCII (Unicode) values .

So to compare "3" to "200", you first compare "3" to "2"
since "3" (ASCII value 51) is greater than "2" (ASCII value 50),
you are done. "3" > "200"

It probably makes more sense if you compare "B" to "ABC"
"B" > "ABC" makes intuitive sense

It gets slightly crazy if you compare "b" to "B"
"b" is greater than "B".
Why? The ASCII value of "b" is 98, but "B" is 66
