Logical operators examples

There are three logical operators in Python:

  • and - condition1 and condition2 is True
    if both condition1 and condition2 are True
  • or - condition1 or condition2 is True
    if either condition1 or condition2 (or both) are True
  • not - not condition1 is True if condition1 is False

Order of operations

  • parentheses evaluated first
  • then not
  • then and
  • then or
