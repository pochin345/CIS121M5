Constants / Variables / Types

Values in any programming language can be fixed (constants)
or able to be changed (variables)

Both constants and variables have a type associated with them

Some values store text data (called string in Python).

Some can store numbers. We'll look at two kinds of numbers:

  • Whole numbers, integers (called int in Python)
  • Numbers with decimal points (called float in Python)

The input function produces string output
To convert to integer, use the int function
To convert to a number with a decimal, use the float function

Examples:

  • name = input("Please enter your name")
  • age = int(input("Please enter your age"))
  • price = float(input("Please enter the item's price"))
