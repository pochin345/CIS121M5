Standard Input

Programs that have no user interaction are pretty boring.
The standard way to prompt the user for a response in Python is input

The format of the input command is input(prompt)
where prompt is what the computer displays
so the user knows what data to enter

Task

Ask the user what comedy troupe inspired Python's name
and then display the name of that troupe

In case you don't know, the answer is Monty Python

Note:
the + in the output concatenates the two strings:

  • the literal "The comedy troupe that inspired Python was"
  • and the variable comedy_troupe
